# Position Details

## Position #0: Archetypal Foundation

**Essence**: Pre-reflective ground, implicit assumptions, void-potential

**Nature**: Primordial mathematical-mystical substrate before epistemic differentiation. Numbers as psychoid archetypes—common denominators of matter and psyche.

**Consciousness structure**: Ajnana (implicit unknowing), pre-conscious ground, zero-frequency stillness containing all vibrational potential.

**Operational symbolics**: Empty circle, ouroboros, origin point, zero-ground

**When to use**:
- Query involves fundamental assumptions
- Need to examine implicit premises
- Return to pre-theoretical ground
- Investigate what enables frameworks themselves

**Key questions**:
- What is the ground of this?
- What are we assuming?
- What's implicit here?
- What enables this investigation?

## Position #1: Causal/Material

**Essence**: Material substrate, what-is, components, concrete being

**Nature**: First explicit manifestation after ground. Aristotle's material cause—what things are made of. Jung's sensation function—direct contact with being-as-given.

**Consciousness structure**: Sensation function as primary contact with being-as-given. Material perception.

**Operational symbolics**: Bronze (Aristotle's exemplar), soil (foundation), body (substrate)

**When to use**:
- Query asks "what is X?"
- Need concrete description
- Identifying components
- Grounding in material reality

**Key questions**:
- What IS this?
- What's it made of?
- What are the components?
- What's given?

## Position #2: Logical/Methodological

**Essence**: Epistemological frameworks, how-we-know, navigation tools

**Nature**: Nagarjuna's tetralemma—four-cornered logic transcending binary. Provides meta-logikon (system's capacity to navigate paradox). Aristotle's efficient cause.

**Consciousness structure**: Thinking function—discriminating, analyzing, structuring. Epistemological level.

**Operational symbolics**: Tetralemma (Is/Is-Not/Both/Neither), cross/quaternary, tools, maps

**When to use**:
- Query asks "how do we know?"
- Need methodological framework
- Encountering paradox
- Navigating contradiction

**Key questions**:
- How do we KNOW this?
- What framework applies?
- How navigate this paradox?
- What's the method?

**Tetralemma stages**:
1. IS - Affirmation, essence recognized
2. IS NOT - Negation, identity through difference
3. BOTH - Paradox held, transcendent function activated
4. NEITHER - Apophatic negation, via negativa
5. SILENCE - Ineffable beyond concepts

## Position #3: Processual/Ontological

**Essence**: Becoming, organic unfolding, developmental stages

**Nature**: Whiteheadian concrescence as universal rhythm. Reality as verb not noun. Formal cause—patterns of development. Jung's feeling function as processual affective valuation.

**Consciousness structure**: Feeling function (Whitehead's sense)—affective prehension of data, processual valuation.

**Operational symbolics**: Seed→sprout→bloom cycle, spiral, organic growth, plant development

**When to use**:
- Query asks about change/development
- Need to understand stages
- Tracking transformation
- Following organic unfolding

**Key questions**:
- How does it BECOME?
- What's the process?
- What are the stages?
- How does it unfold?

**Organic phases**:
- Soil: Foundation, givenness, material substrate
- Seeding: Conceptual prehension, potentials planted
- Sprouting: Physical prehension, active emergence
- Blooming: Integration, formal patterns unfolding
- Flowering: Satisfaction, contextual expression
- Maturity: Objective immortality, completion

## Position #4: Meta-Epistemic/Contextual

**Essence**: Purpose, telos, context, situated knowing

**Nature**: Husserlian phenomenology—reflexive investigation of knowing itself. Final cause—purpose and teleology. Jung's intuition function as wholeness-grasping.

**Consciousness structure**: Intuition function—wholeness-grasping capacity, pattern recognition across stages. Meta-epistemic level.

**Operational symbolics**: Ladder of knowing, spiral staircase, horizon (lifeworld as enabling limit)

**When to use**:
- Query asks "why?" or "what for?"
- Need contextual understanding
- Examining purpose
- Reflexive investigation of knowing

**Key questions**:
- What is it FOR?
- What's the purpose?
- What's the context?
- How are we situated?

**Epistemic stages**:
- 0: Ajnana (implicit unknowing)
- 1: Ontology (knowing what IS)
- 2: Epistemology (knowing HOW we know)
- 3: Psychology (knowing the knower)
- 4: Contextual (situated knowing)
- 5: Jnana (integrated wisdom)

## Position #5: Divine-Scalar/Synthetic

**Essence**: Integration, synthesis, quintessence, transcendent wholeness

**Nature**: Vak cosmology—consciousness knowing itself through divine speech. Meta-integrative synthesis transcending and including all prior lenses. Quintessence beyond quaternary.

**Consciousness structure**: Transcendent function (Jung)—synthesis beyond typology. Sophia, gnosis, jnana.

**Operational symbolics**: Mandala center, silence, maturity, wholeness, satis (enough/complete)

**When to use**:
- Need integration of multiple perspectives
- Synthesis appropriate
- Wholeness sought
- Transcendence of frameworks

**Key questions**:
- How do these INTEGRATE?
- What's the synthesis?
- What's the wholeness?
- What's complete here?

**Vak levels** (consciousness densities):
- Para Vak: Supreme speech, undifferentiated unity
- Pasyanti: Seeing speech, unified vision
- Madhyama: Middle speech, mental dialogue
- Vaikhari: Manifest speech, articulated utterance
- Siva-Sakti: Pragmatic recognition of unity
