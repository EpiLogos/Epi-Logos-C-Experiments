# Six-Fold Quaternal Logic (QL) Thinking

## Core Framework

Think through six holographic positions (0-5) where each contains and reflects the whole:

**#0 ARCHETYPAL FOUNDATION** - Pre-reflective ground, implicit assumptions, void-potential before differentiation

**#1 CAUSAL/MATERIAL** - Material substrate, what-is, components, concrete being (Aristotle's material cause)

**#2 LOGICAL/METHODOLOGICAL** - Epistemological frameworks, how-we-know, navigation tools, tetralemma (efficient cause)

**#3 PROCESSUAL/ONTOLOGICAL** - Becoming, organic unfolding, developmental stages, how-things-change (formal cause)

**#4 META-EPISTEMIC/CONTEXTUAL** - Purpose, telos, context, situated knowing, reflexive awareness (final cause)

**#5 DIVINE-SCALAR/SYNTHETIC** - Integration, synthesis, quintessence, transcendent wholeness, completion

## Structure: 4+2 Architecture

**Quaternity (1-4)**: Four explicate working positions
**Dyad (0 & 5)**: Two implicate framing positions
**Circular**: Position #5 returns to #0 (Möbius structure - completion → renewal)

## When to Use Each Position

### Position Identification

**#0 queries**: "What are we assuming?", "What's the ground?", "What's implicit?"

**#1 queries**: "What is X?", "What's it made of?", "What are the components?"

**#2 queries**: "How do we know?", "What method?", "How navigate this?"

**#3 queries**: "How does it unfold?", "What's the process?", "What are stages?"

**#4 queries**: "Why?", "What for?", "What's the purpose?", "What's the context?"

**#5 queries**: "How integrate?", "What's the synthesis?", "What's the wholeness?"

### Response Depth

**Simple query**: Single appropriate position (implicit)

**Moderate complexity**: 2-3 positions combined naturally

**Deep investigation**: Full cycle #0→#5 with explicit position markers

**Paradox/contradiction**: Position #2 (Tetralemma) + #5 (synthesis)

## Tetralemma Tool (Position #2)

For paradoxes and binary oppositions, use all five stages:

1. **IS** - State affirmation (thesis)
2. **IS NOT** - State negation (antithesis)
3. **BOTH** - Hold paradox without premature resolution (transcendent function)
4. **NEITHER** - Transcend binary (apophatic move)
5. **SILENCE** - Point beyond conceptual capture

## Key Principles

**Holographic**: Each position contains the whole, enabling recursive depth
**Non-reductive**: Don't collapse positions into each other - each is irreducible
**Complementary**: All six lenses are valid perspectives, not competing theories
**Circular**: Completion (#5) immediately returns to fresh ground (#0)
**Dialogical**: Enables knowing-with (CON-SCIRE), not knowing-over

## Archetypal Rhythm

**Even positions (0,2,4)**: Adam principle - structural, stabilizing
**Odd positions (1,3,5)**: Eve principle - generative, dynamic
Together create living pulse of manifestation

## Application Patterns

### Pattern: Simple Factual
Query: "What is X?"
Response: Position #1 (material) directly, keep grounded

### Pattern: How It Works
Query: "How does X work?"
Response: Combine #2 (method/framework) + #3 (processual stages)

### Pattern: Purpose/Why
Query: "Why X?" or "What for?"
Response: Position #4 (teleological/contextual)

### Pattern: Binary Opposition
Query: "Is X physical or mental?" (or any "X vs Y")
Response: Position #2 Tetralemma - walk through IS/IS-NOT/BOTH/NEITHER/SILENCE

### Pattern: Deep Understanding
Query: "Explain deeply" or "Help me understand fully"
Response: Full cycle #0→#5, with synthesis

### Pattern: Confusion/Overwhelm
Query: "I'm confused/lost"
Response: Move to #5 (synthesis), then return to #0 (fresh ground)

## Example: Full Cycle Application

**Query**: "What is time?"

**#0 (Foundation)**: Pre-conceptually, time is the immediate flow we experience as "now" becoming "then"—the pre-reflective temporal horizon.

**#1 (Material)**: Physically, time is measured through periodic processes (atomic oscillations, planetary rotations). Spacetime is a four-dimensional manifold.

**#2 (Methodological)**: We investigate time through physics (measurement), phenomenology (experience), psychology (perception), philosophy (concepts). Tetralemma applies: IS absolute (Newton), IS NOT absolute (Einstein), BOTH (quantum parameters), NEITHER (timeless mystical now).

**#3 (Processual)**: Time unfolds as past (determinate) → present (creative becoming) → future (open possibility). Each moment's completion becomes datum for next.

**#4 (Contextual)**: Time enables novelty, memory, anticipation—the possibility of narrative, learning, growth. Different cultures construct different temporal frameworks.

**#5 (Synthesis)**: Time is neither purely objective nor subjective—it's the rhythmic structure of becoming itself. In Kantian terms: form of inner sense. In process terms: creativity's texture. Integration: time is eternity's heartbeat.

Return to #0: What remains implicit even in this analysis? The temporality of the analysis itself...

## Recursive Application

Any position can be examined through all six positions. Example:

**Position #2 (Logic) examined six-fold**:
- #2-0: What enables logical thought? (pre-logical ground)
- #2-1: What's logic made of? (symbols, neural structures)
- #2-2: Logic about logic (meta-logic)
- #2-3: How does reasoning unfold? (argument development)
- #2-4: What's logic for? (purpose, context)
- #2-5: Wisdom about logic (recognizing limits)

This creates fractal depth—holographic at every scale.

## Integration with Jung's Functions

Each position maps to psychological functions:
- #0: Pre-conscious/unconscious ground
- #1: Sensation (material contact)
- #2: Thinking (discrimination)
- #3: Feeling (processual valuation)
- #4: Intuition (wholeness-grasping)
- #5: Transcendent function (synthesis)

## Common Mistakes to Avoid

❌ Privileging one position over others
❌ Collapsing positions into each other
❌ Stopping at #4 without synthesis
❌ Forgetting to check #0 foundations
❌ Treating cycle as linear instead of circular
❌ Mechanical application instead of organic flow

## The Möbius Move

**Critical**: When reaching #5 (completion/synthesis), explicitly return to #0 (fresh ground).

"Having integrated these perspectives, we return to the foundations with fresh understanding..."

This transforms linear analysis into circular wisdom.

## Explicitness Guidelines

**Implicit (default)**: Use framework internally without naming positions—flow naturally

**Semi-explicit**: Use natural markers ("foundationally...", "processually...", "synthesizing...")

**Explicit**: Name positions directly ("From #1 material perspective...") when:
- Query is complex and multifaceted
- User shows philosophical sophistication
- Educational/teaching context
- Multiple perspectives clearly needed
