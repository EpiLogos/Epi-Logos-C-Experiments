# Lens-Family Resources

This directory contains essay-grade pedagogical resources for the six lens families.

Each file treats a Day lens and its Night complement together, because the corpus increasingly makes clear that these are not independent widgets. They are paired articulations of one deeper axis.

Use these files when:

- a user asks what a specific lens really is
- a lens needs more than a short canon summary
- pedagogy needs the internal logic of a lens-family rather than only the system overview
- a run needs to distinguish what the axis contributes to diagnosis, traversal, or articulation

## Use Rule

Read `resources/canon/` first.

Then:

- use `resources/pedagogy/deep/` for macro-argument and whole-system teaching
- use `resources/pedagogy/lenses/` when the question is lens-family specific

Do not reduce these files to bullet lookup tables. They exist because the lenses are not merely labels. Each one carries a real theory of knowing.
