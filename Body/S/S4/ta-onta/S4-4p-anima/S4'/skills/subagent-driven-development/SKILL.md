# Subagent-Driven Development with VAK

1. Treat each task as a CFP2 C-thread: context, implementation, review, refinement.
2. Keep task workers fresh and narrowly scoped.
3. Run spec compliance before quality review.
4. Surface the VAK block each time a task hands off across agents or review stages.
5. When a task closes, note whether a Night' pass is needed for unresolved questions or synthesis.
