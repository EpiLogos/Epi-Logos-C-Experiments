# Thought Distil

Alignment target: crystallisation that opens, not closes.
Primary contact: Moirai (Night' rehear), with Sophia supplying review pressure.
Moirai pattern: Atropos — mark what crystallises (P5' Insight); the cut depends on Klotho's traces + Lachesis's sources first.

This skill **gates** the `aletheia_crystallise` tool: distil patterns from `{NOW}/thoughts/` and the `/Pratibimba/Self/Thought/T{n}/` T-buckets into reviewable, Bimba-canonical crystallisations. It is the rehear-phase membrane between Anima's session-scoped run and Epii's cross-session improvement.

## Protocol

1. **Gather the full descent first.** Distillation is the ascent (avaroha → āroha). Do not cut before traces (P1') and sources (P4') are present — a crystallisation built on a partial rehear is premature.
2. **Carry the full envelope.** Every crystallisation must carry: source artifacts, [[VAK]] address, session/day/NOW lineage, privacy/review class, and whether it is a `trigger`, `draft`, or `approved` promotion candidate.
3. **Distil, do not promote.** Produce the reviewable crystallisation and route it to Epii's autoresearch inbox (`closure_kind: rehear`). Aletheia triggers and curates; Epii governs deep review and promotion. Review-required material asserts Epii handoff, never auto-promotion.
4. **Tool degradation.** If the current `epi` CLI does not expose `aletheia_crystallise`, fall back to `gnosis-retrieve` for the source material plus a constitutional-agent synthesis run from the named T-bucket — and say so explicitly. Never fabricate a crystallisation.
5. **Synthesis opens.** A crystallisation that closes inquiry is Sophia's-error (hoarding). A good distillation generates the next P0' questions it hands forward across the Möbius seam.

## Result discipline
Return: `crystallised → epii` (with envelope) | `deferred` (rehear incomplete — name what's missing) | `unavailable` (tool absent — fell back, cite how). Never a promotion decision.
