# Day Night Pass

Use this skill when Anima or Psyche must preserve continuity across a session boundary.

## Contract

- Capture the current task, live subtasks, artifacts, blockers, and next action.
- Distinguish executable follow-up from reflective review.
- Deposit validation or meaning-review items to Epii instead of resolving them inside Anima.
- Preserve `NOW` and `Day` references when present.

## Output Shape

```json
{
  "current_task": "Phase 5 S4 agent access",
  "completed": [],
  "open_threads": [],
  "next_action": "Run agent invocation proof",
  "epii_inbox_items": []
}
```
