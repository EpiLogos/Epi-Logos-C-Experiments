## How to reproduce the fixture

```bash
# Verify the fixture is byte-stable against current portal-core code:
cargo test --manifest-path Body/S/S0/portal-core/Cargo.toml --test track_01_t0_baseline_fixture

# Re-bless after an intentional canonical-shape change:
BASELINE_FIXTURE_BLESS=1 cargo test --manifest-path Body/S/S0/portal-core/Cargo.toml --test track_01_t0_baseline_fixture
```

The bless mode is **only** for intentional schema evolution. The default mode panics on drift.
