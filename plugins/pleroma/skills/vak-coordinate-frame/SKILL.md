# VAK Coordinate Frame Reference

Complete grammar for the 6-layer VAK coordinate system (S4-0′ through S4-5′).

## CPF — S4-0′: Context Processing Frame
| Code | Mode | Description |
|------|------|-------------|
| `(00/00)` | Dialogical | User-engaged; Ouroboros / Nous |
| `(4.0/1-4.4/5)` | Mechanistic | Execution; Ralph mode / Anima |

## CT — S4-1′: Content Type
| Code | Name | Domain |
|------|------|--------|
| CT0 | Relational | Dialogue, discovery, relationship |
| CT1 | Definitional | Specs, architecture, research |
| CT2 | Operational | Code, tests, scripts |
| CT3 | Pattern | Analysis, debugging, refactoring |
| CT4 | Contextual | Planning, coordination |
| CT5 | Integrative | Synthesis, reflection |

## CP — S4-2′: Context Position
| Code | Day | Night′ |
|------|-----|--------|
| 4.0 Ground | What do we have? | P0′ Questions: What don't we know? |
| 4.1 Definition | What must be true? | P1′ Traces: What evidence exists? |
| 4.2 Operation | What is being done? | P2′ Challenges: What blocks us? |
| 4.3 Pattern | What shape does it take? | P3′ Patterns: What repeats? |
| 4.4 Context | Where/when? | P4′ Discovery: What sources inform? |
| 4.5 Integration | What was produced? | P5′ Insight: What crystallizes? |

Night′ traversal: 4.5 → 4.4 → 4.3 → 4.2 → 4.1 → 4.0
Möbius return: P5′ Insight → P0′ Questions (cycle resumes)

| CF Code | Agent | Role |
|---------|-------|------|
| `(00/00)` | Nous | Brainstorming, Ouroboros dialogical |

## CFP — S4-4′: Context Frame Position
| Code | Name | Description |
|------|------|-------------|
| CFP0 | Base | Single-agent, sequential |
| CFP1 | P-Thread | Parallel: different tasks → N agents |
| CFP2 | C-Thread | Chain: one task → sequential agents |
| CFP3 | F-Thread | Fusion: same task → N agents → merge |
| CFP4 | L-Thread | Lemniscate: Night′ pass |
| CFP5 | B-Thread | Background/async |
| Z | Z-Thread | Autonomous autopilot (no user prompts) |
