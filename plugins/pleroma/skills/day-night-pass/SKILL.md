# Day/Night′ Pass

Traverse the quaternal cycle in Day or Night′ mode, or both.

## Day Pass (Torus Forward)

Walk CP 4.0 → 4.5 in sequence:

| Position | Question | Agent |
|----------|----------|-------|
| CP 4.0 Ground | What do we have? | Nous / Anima |
| CP 4.1 Definition | What must be true? | Logos |
| CP 4.2 Operation | What is being done? | Eros / Anima |
| CP 4.3 Pattern | What shape does it take? | Mythos |
| CP 4.4 Context | Where/when does it apply? | Anima |
| CP 4.5 Integration | What was produced? | Sophia |

## Night′ Pass (Lemniscate — orthogonal inquiry)

Walk P5′ → P0′ in sequence (reverse, Night′ positions):

| Position | Question | Moira |
|----------|----------|-------|
| P5′ Insight | What crystallizes? | Atropos → Sophia |
| P4′ Discovery | What sources inform? | Lachesis → Anima |
| P3′ Patterns | What repeats? | (Anima/Mythos) |
| P2′ Challenges | What blocks us? | Klotho → Eros |
| P1′ Traces | What evidence exists? | Klotho → Eros |
| P0′ Questions | What don't we know? | Nous |

Night′ is NOT a backward review of Day. It is orthogonal inquiry that surfaces what Day missed.

## Möbius Return

P5′ Insight does not close the cycle — it generates P0′ Questions for the next torus.
Sophia hands the crystallized insight back to Nous as new ground.

`P5′ (Sophia) → P0′ (Nous)` = the Möbius return signal.

## Invocation

- **Day only**: use for forward synthesis after a clear task specification.
- **Night′ only**: use after a Day pass to interrogate assumptions (CFP4 L-Thread).
- **Day + Night′**: full pass for complex, high-risk, or ambiguous work.
