## Description

CFP thread types (Base/P/C/F/L/B/Z) - the formal patterns of agent workflows.

## Operations

```python
class ThreadPatternAPI:
    def create_thread(thread_type: str, task: Task) -> ThreadID
    def get_thread_info(thread_id: ThreadID) -> ThreadInfo
    def list_active_threads() -> List[ThreadInfo]
    def coordinate_threads(thread_ids: List[ThreadID]) -> CoordinationResult
    def evolve_thread(thread_id: ThreadID, target_type: str) -> EvolutionResult
```
