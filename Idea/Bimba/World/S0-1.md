# S0.1: Atomic Commands

> **QL Position**: #1 - Definition/Material
> **Parent**: S0 (Terminal)

## Description

The material atoms of terminal interaction - indivisible commands that form the basis of all scripting.

## Constitutional Sanskrit

**SENT** - Commands receive purpose from user intent

## Core Commands

- `cat` - read content
- `echo` - write content
- `ls` - list directory
- `read` - read input
- `cd` - change directory
- `pwd` - print working directory
- `mkdir` - create directory
- `rm` - remove file/directory
