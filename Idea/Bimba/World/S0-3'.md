> **QL Position**: #3 - Pattern/Form

## Description

API for script pattern operations - functions, loops, conditionals, and reusable structures.

## Operations

```python
class ScriptAPI:
    def create_function(name: str, body: List[str]) -> Function
    def execute_function(func: Function, args: List[str]) -> ScriptResult
    def for_loop(iterable: List, body: List[str]) -> LoopResult
    def while_loop(condition: str, body: List[str]) -> LoopResult
    def if_statement(condition: str, true_body: List[str], false_body: List[str]) -> Result
    def assign_variable(name: str, value: str) -> None
    def command_substitution(command: str) -> str
```
