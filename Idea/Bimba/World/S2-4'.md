## Garland Harmony Principle

**The property IS the relationship, the relationship IS the property:**

```cypher
// Property derives from relationships
MATCH (n)-[r]->(m)
SET n.${r.type}_count = count(r)

// Relationship derives from property
MATCH (n)
WHERE n.has_reflection = true
CREATE (n)-[:REFLECTS]->(m)
```
