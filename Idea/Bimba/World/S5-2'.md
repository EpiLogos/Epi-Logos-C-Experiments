# S5.2': Learning Extract API

> **QL Position**: #2 - Operation/Process
> **Parent**: S5' (Reflection API)
> **Constitutional**: SPARSHA - Contact, learning touches ground

## Description

Extract learning by Sanskrit constitutional stage. SPARSHA - the moment learning makes contact with reflection ground, wisdom touches consciousness.

## Operations

```python
class LearningExtractAPI:
    def extract_learning(period_id: str) -> ConstitutionalLearning
    def extract_sent_learning(synthesis: Synthesis) -> SentStage
    def extract_sparsha_learning(synthesis: Synthesis) -> SparshaStage
    def extract_rupa_learning(synthesis: Synthesis) -> RupaStage
    def extract_rasa_learning(synthesis: Synthesis) -> RasaStage
    def extract_scent_learning(synthesis: Synthesis) -> ScentStage
    def validate_constitutional_integrity(learning: ConstitutionalLearning) -> bool
```

## Constitutional Meaning

**SPARSHA** - Contact. The stage where perception touches consciousness, where the raw data of experience becomes felt knowledge. Learning is not abstract - it is the touch of wisdom on the mind.

## Constitutional Learning Structure

```python
@dataclass
class ConstitutionalLearning:
    sent: SentStage  # Divine emanation
    sparsha: SparshaStage  # Contact/work completed
    rupa: RupaStage  # Form/patterns recognized
    rasa: RasaStage  # Taste/complexity experienced
    scent: ScentStage  # Pattern/quintessence distilled

@dataclass
class SentStage:
    stage: str = "Divine emanation"
    learning: List[str]  # Top 3 key learnings
    constitutional: str  # Purpose that emerged

@dataclass
class SparshaStage:
    stage: str = "Contact"
    learning: List[str]  # Work completed
    constitutional: str  # What work touched ground

@dataclass
class RupaStage:
    stage: str = "Form"
    learning: List[str]  # Patterns recognized
    constitutional: str  # What forms took shape

@dataclass
class RasaStage:
    stage: str = "Taste"
    learning: List[str]  # Context complexity experienced
    constitutional: str  # What complexity was tasted

@dataclass
class ScentStage:
    stage: str = "Pattern"
    learning: str  # Quintessence statement
    constitutional: str  # The thread holding this period together
```
