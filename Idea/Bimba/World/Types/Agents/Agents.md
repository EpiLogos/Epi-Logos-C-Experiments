# Agents Type MOC

## Scope

[[Agents]] indexes agent identities, constitutional functions, carrier roles, and runtime agent embodiments across [[S4]] / [[S4']] and [[S5]] / [[S5']].

## What Belongs Here

- Constitutional agents: [[Nous]], [[Logos]], [[Eros]], [[Mythos]], [[Psyche]], and [[Sophia]].
- Ta-onta carriers: [[Khora]], [[Hen]], [[Pleroma]], [[Chronos]], [[Anima]], and [[Aletheia]].
- Return and review embodiments such as [[Epii]].

## What Does Not Belong Here

- Unbounded provider personas or ad hoc chat roles.
- Tool permissions without an agent/capability boundary.
