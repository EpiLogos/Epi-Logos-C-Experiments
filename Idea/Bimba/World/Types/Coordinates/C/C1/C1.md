# C1 Forms And Templates

## Scope

[[C1]] holds form and template typology. It names the reusable patterns by which markdown artifacts, definitions, prompts, task specs, pattern notes, daily/NOW files, and thoughts become authored.

## What Belongs Here

- Form and definition classes.
- Ideal markdown forms.
- [[CT]] template law under `C1/C1'/CT/**`.
- Template source pointers used by Hen.

## What Does Not Belong Here

- Concrete entity candidates; use [[C2]].
- Diagrams and canvases as process forms; use [[C3]].
- Type/folder authority; use [[C4]].

## Semantic Subfolders

- `Forms-And-Templates/Forms`
- `Forms-And-Templates/Definitions`
- `Forms-And-Templates/Ideal-Markdown-Forms`
- `C1'/CT/**` for CT0-CT5 template law.

## Hen Use

Hen classifies template-producing and form-authoring artifacts through C1, then applies CT-specific validation.
