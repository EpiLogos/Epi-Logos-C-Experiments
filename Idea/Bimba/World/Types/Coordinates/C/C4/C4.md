# C4 Types Contexts MOCs

## Scope

[[C4]] holds type-authority typology: folder MOCs, context frames, type authorities, Dataview/query indexes, and residency classes.

## What Belongs Here

- Same-name `.md` / `.canvas` MOC authority patterns.
- Folder-based type authorities.
- Context-frame and residency-class classifications.
- Dataview/query index forms for type navigation.

## What Does Not Belong Here

- Templates themselves; use [[C1]].
- Entity/property definitions; use [[C2]].
- Diagrams as visual process forms; use [[C3]].

## Semantic Subfolders

- `Types-Contexts-MOCs/Type-Authorities`
- `Types-Contexts-MOCs/Context-Frames`
- `Types-Contexts-MOCs/Folder-MOC-Patterns`
- `Types-Contexts-MOCs/Dataview-Indexes`
- `Types-Contexts-MOCs/Residency-Classes`

## Hen Use

Hen uses C4 to decide whether a folder is a real type authority, a container, or an incubation folder. A C4 authority must obey the MOC index rule.
