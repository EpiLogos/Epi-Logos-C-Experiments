# I-Ching

## Definition

[[I-Ching]] is **64 hexagrams as 64 codon-quaternions**, projected onto a tetralemma-native line algebra. Where [[Tarot]] folds the codon space into 80 quaternion-points, the Yijing reads the same space *line-wise*, exposing the 6/7/8/9 line values as direct [[Nucleotide]] picks.

## Structure

**64 hexagrams ≡ 64 codon-quaternions.** Each hexagram is a stack of 6 lines, each line a tetralemma-native [[Nucleotide]] state:

- **6 = Old-Yin = A** (Cups / Water).
- **7 = Young-Yin = C** (Pentacles / Earth).
- **8 = Young-Yang = G** (Swords / Air).
- **9 = Old-Yang = T** (Wands / Fire).

**384-line-change operator graph:** 64 hexagrams × 6 lines = 384 line-positions, decomposing as **360 dynamic degree-nodes + 24 palindromic backbone-nodes** at the 15° lattice spokes.

## QL Mapping

- **Lines bottom → top map to [[P0]] → [[P5]]** — line 1 = Truth/Ground; line 6 = Image/Synthesis.
- **Upper trigram = mental-pole trinity** ([[P3]]–[[P5]]); **lower trigram = physical-pole trinity** ([[P0]]–[[P2]]).
- **Nuclear hexagram** (lines 2–5 recombined into a new 6-line stack) = the **[[P4]] lemniscate sub-reading** — where the lemniscate anchor fires inside the reading itself.
- **Complementary pairs:** lines 1↔6 → [[Square A]]; lines 2↔5 → [[Square B]]; lines 3↔4 → [[Square C]].

## Compass and Element

- **Dual compass:** Early Heaven (Fu Xi) arrangement + Later Heaven (King Wen) arrangement, read as two simultaneous orderings of the same 64-codon space.
- **Wu Xing interactions:** elemental-cycle reading layered over the line-by-line nucleotide picks (per [[L2']] alchemical-elemental).
