# C0 Source Ground

## Scope

[[C0]] holds source-ground typology: coordinate roots, ontology roots, source namespaces, and the folder-ground from which forms become classifiable.

## What Belongs Here

- Coordinate root definitions.
- Ontology source namespaces.
- Source/root anchors used by Hen residency and graph-promotion evidence.

## What Does Not Belong Here

- Template families; use [[C1]] / [[CT]].
- Entity/property/tag definitions; use [[C2]].
- MOC/canvas process forms; use [[C3]].

## Semantic Subfolders

- `Source-Ground/Coordinate-Roots`
- `Source-Ground/Ontology-Roots`
- `Source-Ground/Source-Namespaces`

## Hen Use

Hen uses C0 as the ground check: does this artifact have a source condition and coordinate root before it is classified further?
