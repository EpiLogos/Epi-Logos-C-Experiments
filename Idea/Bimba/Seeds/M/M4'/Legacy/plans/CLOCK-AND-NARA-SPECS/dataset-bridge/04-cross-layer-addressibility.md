## §1 — Critical Corrections (Confirmed by Subagents)

These correct prior spec assumptions and must propagate everywhere:

### 1.1 M1/M2/M3 Context Labels in Generation Events ≠ Subsystem References

The `context` field on `#3-3` generation event nodes (`M1-H{n}`, `M2-H{n}`, `M3-H{n}`)
refers to **three internal matrix application paths within M3 Mahamaya**, NOT to the
M1/M2/M3 subsystems. Do not read these as cross-branch pointers.

### 1.2 Tarot Suit Indexing is 1-Based

Mahamaya tarot coordinates: `#3-4-1` = Wands/Fire, `#3-4-2` = Cups/Water,
`#3-4-3` = Swords/Air, `#3-4-4` = Pentacles/Earth.
Any code or spec assuming 0-based suit indexing (`#3-4-0` = Wands) is **wrong**.

### 1.3 7-State Non-Dual Codons: Eigenspace Collapse, Not "1 + Groups"

7-state cards: the 7th state is a **degenerate eigenstate** (pair1 = pair2 collapsed),
not a separate "non-dual" orientation alongside 3+3. The 7 states = 6 active orientations
+ 1 eigenspace-degenerate anchor. Spec 13 description needs updating.

### 1.4 Three Pairing Matrices Are Not Discrete Nodes (Yet)

The Clifford involutions exist only in the binary structure of generation events, not as
addressable `#3-*-matrix-{n}` nodes. They must be reified as `#3-2-matrix-{1,2,3}`.

### 1.5 Shadow Decans: M2 Is Primary, M3 Is Expression

The decan shadow double-cover exists as **dedicated light/shadow variant nodes at `#2-3`**
(Parashakti, 108 nodes: 36 primary + 36 light + 36 shadow). The `reversedMeaning` on M3
pip cards is the **tarot expression** of this M2 structural fact. M2 = ground; M3 = mirror.

### 1.6 Rotational State Nodes Are Not Persisted

The 512 `RotationalState` nodes (64 codons × 8 states) do not exist in the current dataset.
They are generated by `rotational_state_protocol.txt`. Phase 2 task: materialize and persist.

### 1.7 Planet Dataset Has 7 Classical Planets (Missing Outer Planets)

`parashakti-planets.json` has Sun + 6 classical + Earth + 8 chakras. Uranus/Neptune/Pluto
are absent. Target coordinates when resolved: `#2-5-8`, `#2-5-9`, `#2-5-10`.
