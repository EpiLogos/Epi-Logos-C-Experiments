## Canonical Role

[[S3.0']] is the [[P0']] / [[CT0]] ground of [[S3']] protocol truth: versioned connection law, connect-first invariant, canonical error shape, and shared runtime identity before a session becomes projected temporal state. ## Build Contract

Protocol law must be stable enough for [[M']], [[KernelBridge]], [[OmniPanel]], and external clients to reconnect and reason about drift. Any protocol version bump must be mirrored in tests and app clients. The first response must advertise capabilities truthfully; pre-connect traffic must be refused or constrained by typed error semantics. Contract identity must include session/day/NOW affordance without publishing protected profile or journal details.

## Test Obligations

Contract tests must cover method/event advertisement, protocol family identity, connect-first expectations, subscription envelope identity, and silent-fallback sentinels. These tests should run against real serialised shapes, not hand-waved mock structs.
