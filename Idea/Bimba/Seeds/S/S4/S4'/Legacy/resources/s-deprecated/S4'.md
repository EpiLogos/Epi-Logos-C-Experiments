# S4': Agent Runtime API (Moltbot Gateway)

> **QL Position**: #4 - Context/Environment
> **Implementation**: **Moltbot** (personal AI assistant gateway)
> **Gateway**: ws://127.0.0.1:18789

## Description

S4' IS the Moltbot Gateway - the agent runtime that hosts all S3' plugins as extensions, provides session management, hook system, skills loader, and cron scheduler.

## Core Operations

| S4.Y' | Operation | Moltbot Component | Description |
|-------|-----------|-------------------|-------------|
| S4.0' | Session Init | Gateway boot | Create session, establish context |
| S4.1' | Task Define | Message parsing | Route messages, parse tasks |
| S4.2' | Execute | Agent invocation | Run agent with tools |
| S4.3' | Context Frame | Context provision | Provide context to agents |
| S4.4' | Coordinate | Multi-agent | Delegation, coordination |
| S4.5' | Result | Response synthesis | Synthesize output, extract learning |

## Moltbot Components

### Gateway (S4.0'-S4.2')

```
ws://127.0.0.1:18789
├─ session.create    → S4.0' init
├─ session.send      → S4.2' execute
├─ session.state     → S4.3' context
└─ session.stop      → Cleanup
```

### Hook System (S4.3')

```typescript
// Available hooks
registerInternalHook('agent:bootstrap', handler)  // Context injection
registerInternalHook('command:new', handler)      // Custom commands
registerInternalHook('session:start', handler)    // Session lifecycle
registerInternalHook('session:stop', handler)     // Cleanup
registerInternalHook('gateway:*', handler)        // Gateway events
```

### Skills Loader (S4.2')

```
skills/*.md format
├─ Frontmatter with metadata
├─ Prompt/instructions
└─ Loaded at runtime
```

### Cron Scheduler (S4.4')

```json
{
  "job-name": {
    "schedule": { "type": "cron", "cron": "0 4 * * *" },
    "payload": { "kind": "systemEvent", "text": "/command" },
    "sessionTarget": "isolated",
    "enabled": true
  }
}
```

### Session Memory (S4.3')

```
SQLite + sqlite-vec
├─ Session transcripts (JSONL)
├─ Workspace file vectors
├─ Hybrid BM25 + vector search
└─ Ephemeral, per-session
```
