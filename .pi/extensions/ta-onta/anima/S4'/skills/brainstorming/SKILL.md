## VAK Integration Block (Ta Onta extension)

**CF Code:** `(0/1)` or `(00/00)` depending on ambiguity
**CS State:** Day, unless Night' inquiry is explicitly requested
**CPF Mode:** `(00/00)`

# Brainstorming with VAK

1. Ground the request in the current project context before proposing shape.
2. Ask one clarifying question at a time only when a real architectural branch is unresolved.
3. Offer 2-3 approaches with tradeoffs and recommend one.
4. Cover architecture, data flow, error handling, and tests.
5. Keep the result lean enough to implement without inventing new scope.
6. End by stating the provisional VAK coordinate block for the work so Anima can hand off cleanly.

## Output Format

```text
VAK: [task-short-name]
CPF: (00/00)  CT: CT[n][,CT[n]]  CP: 4.[n]
CF: ([code]) -> [Agent]  CFP: CFP[n]  CS: CS[n] / Day
```
