## VAK Integration Block (Ta Onta extension)

**CF Code:** `(5/0)`
**CS State:** Day when proving a claim, Night' when checking what still resists closure
**CPF Mode:** `(4.0/1-4.4/5)`

# Verification Before Completion with VAK

1. Identify the command that proves the claim.
2. Run it fresh.
3. Read the whole output and exit status.
4. State the result honestly.
5. If verification is only partial, mark that as a Night' remainder rather than claiming success.

Night' addition: incomplete validation is not “good enough”; it is unresolved material to be carried into P2'/P5' inquiry.

## Output Format

```text
VAK: [task-short-name]
CPF: (4.0/1-4.4/5)  CT: CT2,CT5  CP: 4.5
CF: ((5/0)) -> Sophia  CFP: CFP0  CS: CS[n] / Day or Night'
```
