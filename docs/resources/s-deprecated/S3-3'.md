## Description

Temporal context API, daily flow coordination, Möbius return - Chronos manages time as both sequence (Chronos) and opportune moments (Kairos).

class ChronosAPI:

- **Period creation**: Daily, weekly, session, etc.
- **Möbius return**: #5 → #0 learning loop
