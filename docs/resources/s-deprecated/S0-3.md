# S0.3: Script Patterns

> **QL Position**: #3 - Pattern/Form
> **Parent**: S0 (Terminal)

## Description

Formal patterns that crystallize ad-hoc commands into reusable structures.

## Constitutional Sanskrit

**RUPA** - Patterns take crystallized form

## Patterns

- Functions (`function_name() { ... }`)
- Loops (`for`, `while`)
- Conditionals (`if`, `case`)
- Variables and assignment
- Parameter expansion
- Command substitution

## Script Types

- Bash scripts (`.sh`)
- Python scripts (`.py`)
- One-liners
- Aliases
